# Stop-CPUStressTest

## SYNOPSIS
Stops a CPUStressTest

## SYNTAX

```
Stop-CPUStressTest [<CommonParameters>]
```

## DESCRIPTION
Stops a stress test started by Start-CPUStressTest

## EXAMPLES

### Example 1
```powershell
PS C:\> Stop-CPUStressTest
```

Stops a running CPU stress test
